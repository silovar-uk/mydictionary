# データ形式

## 完全バックアップ（JSON）

`format: "watashi-jiten-backup"`、`formatVersion: 1`を持つJSON。辞書項目・タグ・設定を含み、完全復元の正本とする。

- 文字コード：UTF-8
- 日時：ISO 8601
- 出会った日：`YYYY-MM-DD`
- 削除済み項目：含む

## CSV

スプレッドシート利用・他サービス移行向け。完全復元は保証しない。UTF-8 BOM付きで出力し、先頭が `= + - @` のセルにはアポストロフィを付ける。

## TXT

1行につき1項目。次の簡易形式を読み込める。

```text
言葉
言葉｜一言メモ
言葉｜一言メモ｜タグ1,タグ2
```
